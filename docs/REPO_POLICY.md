# Repository Policy

Updated: 2026-02-15

## Classification

- Status: `incubating`
- Visibility: `public`
- Purpose: coding experiments and verification with Google Jules.

## Publishing Rules

- Keep private client or employer information out of experiments.
- Strip credentials and identifiers from generated artifacts.

## Branch Governance

- Default branch is `main`.
- Branch protection must enforce pull requests and one approval.
- Force-push and branch deletion on `main` are disabled.

## Maintenance

- Review experiment outputs monthly and archive stale branches.
- Keep README aligned with current experiment scope.
